# resume-builder

Created using Vuejs.

## Builds with the following resume

## Current features

1. Instant preview
2. 1 theme
3. add the to the resume
   - Name
   - Image
   - Address
   - Title
   - Websites (multiple)
   - Email
   - Phone
   - Career Summary
   - Work Experiences (multiple)
     - Title
     - Company
     - Description
     - Period
   - Skills (multiple)
     - Tool name
     - Expertise
   - Education (multiple)
     - Degree
     - Institution
     - Passing year
     - Grade
   - Awards and Certifications
     - Title
     - Institution
   - Interests and Hobbies
4. **Download as pdf**
